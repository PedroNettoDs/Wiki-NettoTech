Projeto AWS – Portfólio no Free Tier

Objetivo

Subir meu portfólio pessoal na AWS utilizando Terraform, aplicando boas práticas de Infraestrutura como Código (IaC).

Entregáveis

  • Criar repositório GitHub do projeto
  • Estruturar Terraform com:
    • Provider configurado
    • EC2 Free Tier
    • Security Groups seguros
    • Elastic IP (se necessário)
  • Automatizar deploy
  • Documentar arquitetura
  • Criar README profissional explicando:
    • Arquitetura
    • Custos estimados
    • Diagrama da solução

Critério de Conclusão

✔ Aplicação online e acessível publicamente
✔ Código versionado no GitHub
✔ Documentação clara e profissional

Projeto Python – Facilitador de Manutenção do AD / ServiceDesk

Objetivo

Desenvolver ferramenta em Python para automatizar tarefas comuns de suporte relacionadas ao Active Directory.

Possíveis Funcionalidades

  • Reset de senha automatizado
  • Desbloqueio de usuário
  • Criação de usuário via formulário
  • Log de auditoria
  • Interface Web simples (Flask/FastAPI)

Diferencial

  • Integração com LDAP
  • Controle de permissões
  • Deploy interno via Docker
